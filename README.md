# Projeto AutoLog - Oficina Mecânica

O **AutoLog** é um sistema desenvolvido para gerenciar informações de veículos em uma oficina mecânica. A plataforma permite o cadastro, edição, remoção e leitura de dados relacionados aos veículos e seus respectivos serviços. O objetivo é facilitar a gestão e acompanhamento do histórico de manutenção dos veículos de forma eficiente e prática.

## Funcionalidades

### Cadastro de Veículos
- **Adicionar Veículo**: Permite cadastrar novos veículos na plataforma, com informações como modelo, marca, ano de fabricação, placa, etc.
- **Edição de Veículos**: É possível editar as informações dos veículos cadastrados caso haja alterações necessárias.
- **Remoção de Veículos**: Caso o veículo não seja mais necessário no sistema, ele pode ser removido.
- **Leitura das Informações**: Visualize todas as informações detalhadas sobre os veículos cadastrados.

### Tabela de Veículos
- **Exibição de Veículos**: A tabela exibe todos os veículos cadastrados com suas informações principais, como modelo, placa e status.
- **Filtros e Busca**: O usuário pode filtrar ou buscar veículos específicos dentro da tabela.
  
### Tecnologias Utilizadas

- **Frontend**:
  - **ReactJS**: Biblioteca JavaScript para construir a interface do usuário.
  - **TypeScript**: Superset do JavaScript que adiciona tipagem estática, garantindo maior robustez e manutenção do código.
  - **Tailwind CSS**: Framework CSS utilitário para uma estilização rápida e responsiva, proporcionando design moderno e clean.
  - **Shadcn UI**: Conjunto de componentes de interface com design focado em sombreamento e efeitos visuais, otimizando a experiência do usuário.
